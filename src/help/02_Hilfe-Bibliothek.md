# Hilfe-Bibliothek

Der Hilfe-Dialog verwaltet **mehrere Markdown-Dokumente** im Ordner `src/help/`. Jede `.md`-Datei wird in der linken Seitenleiste als Eintrag angezeigt.

## Neues Dokument anlegen
1. Im Hilfe-Dialog links unten auf **+ Neu** klicken.
2. Einen Namen eingeben (ohne Endung `.md`).
3. Das Dokument wird angelegt und springt sofort in den Bearbeiten-Modus.

## Reihenfolge steuern
Die Dokumente sind alphabetisch nach Dateiname sortiert. Mit einem Zahlen-Praefix laesst sich die Reihenfolge bequem festlegen:

- `01_Uebersicht.md`
- `02_Hilfe-Bibliothek.md`
- `03_FAQ.md`
- `99_Notizen.md`

## Umbenennen / Loeschen
- **Umbenennen:** linke Sidebar, Button **Umbenennen** - das Dokument behaelt seinen Inhalt.
- **Loeschen:** Button **Loeschen** - Sicherheitsabfrage, danach ist die Datei weg.

## Direkt im Dateisystem
Da alles flache `.md`-Dateien sind, kannst du sie auch im Explorer / Editor pflegen. Beim naechsten Oeffnen des Hilfe-Dialogs werden Aenderungen automatisch erkannt.

## Tipps
- **Suche oben in der Sidebar** filtert die Dokumentliste live nach Namen.
- Bei mehreren Notizen lohnt sich ein eigenes Dokument pro Thema (z.B. "Tastenkuerzel", "FAQ", "Bekannte Probleme").
